# Orlando Pirates Football Club Webpage

This project is a simple HTML and CSS webpage created to showcase information about Orlando Pirates Football Club, one of South Africa's most successful and popular football teams. The webpage provides a brief overview of the club, highlights key facts about its history and identity, and includes a gallery of football-related images.

The project demonstrates fundamental web development skills, including creating a structured HTML document, using headings, paragraphs, lists, images, and applying CSS styling to improve the appearance of the page. CSS was used to customise fonts, spacing, colours, image presentation, and to centre an image on the page.

### Features

* Responsive webpage layout
* Information about Orlando Pirates Football Club
* Unordered list of key facts
* Image gallery with a centred image
* Custom styling using CSS
* Clean and accessible HTML structure

### Technologies Used

* HTML5
* CSS3

### Learning Outcomes

Through this project, I gained practical experience with HTML document structure, semantic elements, image embedding, CSS selectors, text formatting, spacing, and basic webpage styling. The project helped reinforce foundational front-end web development concepts.
